🐉 TMJOs APT Repo

Pacotes oficiais TMJOs assinados com GPG.

Setup

curl -fsSL https://packages.tmjos.com.br/keys/tmjos-archive-keyring.gpg \
  | sudo tee /usr/share/keyrings/tmjos-archive-keyring.gpg > /dev/null
echo 'deb [signed-by=/usr/share/keyrings/tmjos-archive-keyring.gpg] \
  https://packages.tmjos.com.br/ noble main' \
  | sudo tee /etc/apt/sources.list.d/tmjos.list
sudo apt update
sudo apt install tmjos
