🐉 TMJOs Suite

Apps proprietários TMJOs assinados com GPG. Funciona em qualquer Linux Debian-based com apt: Ubuntu, Debian, Linux Mint, Pop!_OS, Kali, Tails, Parrot, elementaryOS, Zorin OS, etc.

Setup

curl -fsSL https://packages.tmjos.com.br/keys/tmjos-archive-keyring.gpg | sudo tee /usr/share/keyrings/tmjos-archive-keyring.gpg > /dev/null
echo 'deb [signed-by=/usr/share/keyrings/tmjos-archive-keyring.gpg] https://packages.tmjos.com.br stable main' | sudo tee /etc/apt/sources.list.d/tmjos.list
sudo apt update

Apps disponíveis

sudo apt install tmjpad    # Editor de texto Rust com session persistence
sudo apt install tmjmenu   # Popup launcher + dock (TMJMenu + TMJDock)
sudo apt install tmjstore  # Software center proprietário
